# 欢迎来到 create-koa-template 👋

使用其他语言阅读: [English](./README.md) | 简体中文

## 介绍

一个快速创建 Koa 服务脚手架的工具。

## 功能

  1. 交互式创建 Koa 服务的脚手架
  2. 支持以下中间件
      - koa-views
      - koa-router
      - koa-static
      - koa-body

## 安装
```bash
npm install koa-create-template -g
```
## 使用
```bash
koa-create-template create projectname
```
